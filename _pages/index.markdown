I'm currently studying Computer Science at the University of Waterloo. I mostly make web things, like this [transient textboard](https://textboard.net). I'm also making an [adorable blob game](https://github.com/dispensablegames/locoroco). I like things that are cute but simple. I like free software and GNU/Linux. I also like mechanical keyboards... a lot.

## Projects

### Blob Game --- [Link to GitHub](https://github.com/dispensablegames/locoroco)

{% include image.html src="https://raw.githubusercontent.com/dispensablegames/locoroco/master/previewfiles/v1.gif" alt="Video of Game" %}

This is a remake of Sony's *LocoRoco*. It uses Lua and LOVE2d. Our game uses `.svg` files for the levels, so anyone can make their own level. It's still in progress!

### Textboard --- [Link to Project](https://textboard.net)

{% include image.html src="/images/textboard1.png" alt="Picture of Textboard Main Page" %}
{% include image.html src="/images/textboard2.png" alt="Picture of Textboard Board Page" %}

I wanted to design my very own online anonymous forum. So I did. It uses Django, PostgreSQL, and vanilla JavaScript. I gave it a retro design to reflect its old-school nature but I also made sure it was responsive and able to work on mobile devices.

### SPaSM Club --- [Link to Project](https://spasm.club)

{% include image.html src="/images/spasm.png" alt="Picture of SPaSM Club Website" %}

I made a website for the Physics and Math Club at my old high school with the Jekyll site generator, hosted with Apache on my own server. Everyone likes the gradient Klein bottle motif that has a different colour on each page. There's even an [application page](https://spasm.club/klein) to generate Klein bottle images with arbitrary colours, made using vanilla JavaScript.
